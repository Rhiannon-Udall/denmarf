Welcome to denmarf's documentation!
===================================

**denmarf** (Density EstimatioN using Masked AutoRegressive Flow) is a python package that provides a ``scikit-learn``-like interface to perform density estimation using `masked autoregressive flow <https://arxiv.org/abs/1705.07057>`_.
